# Recording Light Control

Recording Light Control gives precise control over the Nothing Phone (3)'s recording light.

## Prerequisites

**Root Access**: Tested using KernelSU

**[Shizuku](https://play.google.com/store/apps/details?id=moe.shizuku.privileged.api)**: if Root Access is not available

## Download
