# Recording Light Control

Recording Light Control gives precise control over the Nothing Phone (3)'s recording light.

## Prerequisites

**Root Access**: Tested using KernelSU

**[Shizuku](https://play.google.com/store/apps/details?id=moe.shizuku.privileged.api)**: if Root Access is not available

## Download

You can find the latest APK in the [Releases](https://github.com/Farpathan/Recording-Light-Control/releases) section.


## How to Build from Source:
### Prerequisites
- JDK 17 or higher
- Android SDK (API 35)

### Using Android Studio 
1. Clone the repository
2. Launch Android Studio and open the cloned repository folder
3. Gradle will sync automatically, Wait for it to complete
4. Build or Run it!
